# AI for Traditional Healing

Complete Streamlit demo app:
- SQLite-based users + history
- Secure password hashing with bcrypt
- On-screen 4-digit verification (demo) granting 2 free checks
- Subscription simulation
- Herbs dataset (50+ conditions)
- Works locally and deployable to Streamlit Community Cloud

## Setup & Run

1. Create a folder and save these files:
   - app.py
   - database.py
   - herbs_data.py
   - requirements.txt
   - README.md
